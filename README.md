# Студия маникюра «Линия»

Учебный курсовой проект по курсу «Основы веб-технологий».
Тема: сайт-визитка мастера маникюра (расширено до студии маникюра).

## Структура
- `index.html` — главная страница (hero, услуги, портфолио, видео, отзывы, форма записи)
- `service.html` — страница услуги «Покрытие гель-лак» (страница конкретного объекта)
- `price.html` — страница «Прайс-лист» со всеми услугами и ценами
- `thanks.html` — страница подтверждения после отправки формы
- `css/style.css` — все стили (без библиотек), адаптив 3 брейкпоинта, анимации, доступность
- `img/` — изображения (SVG), фотографии работ (JPG), видео (`promo.mp4`), постер, favicon

## Технологии
HTML5 + CSS3. Без JavaScript и без сторонних CSS/JS-библиотек.

## Запуск
Откройте `index.html` в браузере.
